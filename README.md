# TRYMON GAME ENGINE

Trymon is a Web Game Engine built with a singular focus: **solving specific problems in Web Game Development and providing seamless native export capabilities out-of-the-box.**

## 🎯 Our Core Vision
Developing games for the web brings unique challenges: asset loading strategies, varied browser performance, responsive scaling, and offline support. Trymon is architected from the ground up to solve these web-specific bottlenecks.
More importantly, it provides a stable environment designed to easily export your web games to **Native Platforms** without the usual wrapper-bloat and configuration nightmares.

## 🚀 Features
- **React-Powered Engine Interface**: Robust and predictable state-driven menu and engine hub.
- **Vite & TypeScript Integration**: Modern, blazingly fast development server with strict typings.
- **Multi-Game Hub**: Easily build and manage multiple isolated game prototypes (Snake, Pong, Tetris, etc.) within a unified project structure.

## 🛠 Usage
```bash
# Install dependencies
npm install

# Start the local engine development server
npm run dev

# Build the engine and all games for native export / production
npm run build
```
